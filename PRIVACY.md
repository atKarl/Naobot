Politique de Confidentialité - NaoBot

1. Données collectées
   NaoBot collecte uniquement les données strictement nécessaires à son fonctionnement :
   Votre identifiant unique Discord (User ID).
   Votre pseudonyme sur le serveur.
   Les horodatages (timestamps) de vos messages et réactions.
   Note : Le contenu de vos messages n'est JAMAIS lu ni stocké.
2. Utilisation des données
   Ces données servent exclusivement à :
   Calculer les statistiques d'activité (commande /stats et /top).
   Identifier le "Membre du Mois".
   Gérer l'attribution automatique du rôle "Inactif" après 90 jours.
3. Conservation des données
   Les logs d'activité sont automatiquement supprimés après 365 jours.
   Les données de profil (ID et pseudonyme) sont conservées tant que l'utilisateur est présent sur le serveur.
4. Vos droits (RGPD)
   Vous disposez d'un droit total sur vos données :
   Droit d'opposition : Utilisez la commande /privacy optout pour arrêter immédiatement tout suivi.
   Droit de suppression : Contactez un administrateur pour demander la suppression totale de vos données de la base SQL.
5. Sécurité
   Les données sont stockées dans une base de données SQLite sécurisée sur un serveur privé et ne sont jamais partagées, vendues ou transmises à des tiers.
